# Getting Started

Lith is built for people who want fast capture without giving up structured, long-term note organization.

## Platforms

- iPhone and iPad
- macOS

## What to expect

- Notes are centered on Markdown-style text capture.
- The app is designed to stay local-first for responsiveness and ownership.
- Optional iCloud sync mirrors supported data under your Apple account.
- More advanced workflows are added in stages instead of all at once.

## How to use this guide

- Start with [Notes and Links](notes-and-links.md) for the core workflow.
- Use the feature-area pages to understand how RSS, audio, and search fit into the overall product.
- Use [Privacy and Sync](privacy-and-sync.md) for data-handling expectations.

## Pre-release note

Lith is still evolving. If a page describes a workflow that is not yet available in your current build, treat it as the intended user-facing direction for that feature area rather than a guarantee that every step is already shipped.
