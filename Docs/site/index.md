Lith is a local-first note-taking app for iPhone, iPad, and macOS. It is designed to combine quick capture with longer-term knowledge management without giving up privacy, portability, or native Apple platform UX.

This site is the user-facing documentation surface for Lith. It explains the core concepts that should stay stable across releases and the workflow areas that expand as new features ship.

## Current status

Lith is under active development. Some pages describe capability areas that are planned or still evolving. User-facing changes should update this site in the same issue or pull request that changes the behavior.

## Product principles

- Local-first storage with optional iCloud sync
- Native iOS and macOS experience
- Markdown portability and low lock-in
- Progressive complexity from quick notes to deeper knowledge workflows
