# RSS Inbox

Lith expands beyond manual notes with a review-first RSS workflow.

## Workflow

- Add RSS feeds you trust.
- Review incoming items before they become notes.
- Save only the items worth keeping in your long-term knowledge base.

## Why it matters

The RSS inbox is meant to keep research inputs separate from permanent notes, so your note library stays curated instead of noisy.

## Availability

RSS capture is a planned or evolving capability area. This page describes the workflow direction the app is built around as the feature matures.
